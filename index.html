GIT


    Popular GIT Commands

  • git status : get current git branch
  • git checkout main : switches to main branch
  • git checkout -b feature/feature-name "creates a new branch to add new feature"

    • How to commit changes to Github

  • git status (Check current Branch)
  • git add -A (Commit Changes to current branch)
  • git commit -m "add notes about promotion here" (Sends changes to Github)
  • git pull origin main (pulls down main branch)
  • git push origin feature/feature-name (pushes changes to Github)

    • Merge Branch in Github

  • Navigate to project
  • Navigate to new feature branch
  • Click pull requests page, find and click the "New pull request" button
  • Compare new branch against main, then click Merge pull request button to merge changes with the main branch
